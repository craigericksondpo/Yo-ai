# Contributing

Thanks for your interest in contributing! Please follow these simple guidelines to make the process smooth.

## Getting started
- Create and activate the virtual environment:
  - `python -m venv .venv`
  - `.\.venv\Scripts\Activate.ps1` (PowerShell)
- Install development dependencies if any: `pip install -r requirements-dev.txt` (if provided)

## Code style
- Follow standard Python formatting and linting (Black / flake8 / ruff as applicable).
- Keep changes small and focused.

## Tests
- Add tests for new features or bug fixes.
- Run tests locally before submitting a PR: `pytest`

## Submitting changes
- Fork the repo and open a pull request against the `main` branch.
- Provide a clear description of the change and reference any related issue.
