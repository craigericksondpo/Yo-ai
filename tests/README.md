# yo-ai-assurance-platform \ tests

The "tests" folder contains tests of platform components which generates reports on platform health status on demand, and tests for resolving specific issues.
Some tests, authored by organizations and individuals outside of PrivacyPortfolio and Craig Erickson on the Yo-ai Platform, have been properly attributed and are pulled from the original public repositories when no modifications are needed. Many of these test conformity with the A2A, FastA2A, Starlette, the Google Agent Development Kit, and other frameworks.
